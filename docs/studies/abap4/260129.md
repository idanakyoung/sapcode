- ALV Grid Function Code 전체 설명
    
    (대상: `CL_GUI_ALV_GRID` / Classic ALV 공통 개념)
    
    ### 0. Function Code란?
    
    - ALV 툴바에 있는 **각 버튼의 내부 식별자**
    - 문자열 상수가 아니라 **상수(Constant)** 로 제공됨
    - 예:
        
        ```abap
        cl_gui_alv_grid=>mc_fc_sort_asc
        
        ```
        
    - 주 용도
        1. 특정 버튼 **숨기기(EXCLUDE)**
        2. 사용자 커맨드 처리 (`USER_COMMAND`)
        3. 표준 기능만 선택적으로 사용
    
    ### 1. 기본/점검 계열
    
    ### `MC_FC_DETAIL`
    
    - 선택한 행의 **상세 데이터 팝업**
    - 더블클릭과 유사
    
    ### `MC_FC_CHECK`
    
    - ALV 데이터 **정합성 체크**
    - Editable ALV에서 주로 의미 있음
    
    ### `MC_FC_REFRESH`
    
    - ALV 화면 **다시 그리기**
    - 내부테이블 다시 보여줄 때 사용
    
    ### 2. 로컬 편집(Local Edit) 계열
    
    (Editable ALV에서 의미 있음)
    
    ### `MC_FC_LOC_COPY`
    
    - 선택 셀/행 복사
    
    ### `MC_FC_LOC_COPY_ROW`
    
    - 행 단위 복사
    
    ### `MC_FC_LOC_CUT`
    
    - 잘라내기
    
    ### `MC_FC_LOC_PASTE`
    
    - 붙여넣기
    
    ### `MC_FC_LOC_PASTE_NEW_ROW`
    
    - 붙여넣기 + 새 행 생성
    
    ### `MC_FC_LOC_UNDO`
    
    - 마지막 로컬 변경 취소
    
    ### 3. 행 조작(Row Operation)
    
    ### `MC_FC_LOC_APPEND_ROW`
    
    - 맨 아래에 행 추가
    
    ### `MC_FC_LOC_INSERT_ROW`
    
    - 현재 위치에 행 삽입
    
    ### `MC_FC_LOC_DELETE_ROW`
    
    - 선택 행 삭제
    
    → **Editable ALV + 내부테이블 직접 수정 구조**일 때만 의미 있음
    
    ### 4. 찾기(Find) 계열
    
    ### `MC_FC_FIND`
    
    - 기본 찾기
    
    ### `MC_FC_FIND_MORE`
    
    - 다음 찾기
    
    ### 5. 정렬(Sort) 계열
    
    ### `MC_FC_SORT_ASC`
    
    - 오름차순 정렬
    
    ### `MC_FC_SORT_DSC`
    
    - 내림차순 정렬
    
    ### `MC_FG_SORT`
    
    - 정렬 관련 기능 그룹 전체
    
    ### 6. 필터(Filter) 계열
    
    ### `MC_FC_FILTER`
    
    - 필터 설정
    
    ### `MC_FC_DELETE_FILTER`
    
    - 필터 해제
    
    ### `MC_MB_FILTER`
    
    - 필터 메뉴 버튼 그룹
    
    ### 7. 합계 / 집계(Sum & Total)
    
    ### `MC_FC_SUM`
    
    - 합계
    
    ### `MC_FC_AVERAGE`
    
    - 평균
    
    ### `MC_FC_MINIMUM`
    
    - 최소값
    
    ### `MC_FC_MAXIMUM`
    
    - 최대값
    
    ### `MC_FC_COUNT`
    
    - 개수
    
    ### `MC_MB_SUM`
    
    - 집계 관련 메뉴 버튼 그룹
    
    ### 8. 소계(Subtotal)
    
    ### `MC_FC_SUBTOT`
    
    - 소계 계산
    
    ### `MC_MB_SUBTOT`
    
    - 소계 메뉴 그룹
    
    ### 9. 뷰(View) 변경 계열
    
    ### `MC_FC_VIEW_GRID`
    
    - 기본 그리드 뷰
    
    ### `MC_FC_VIEW_EXCEL`
    
    - Excel 유사 뷰
    
    ### `MC_FC_VIEW_LOTUS`
    
    - Lotus 형식 (구형)
    
    ### `MC_FC_VIEW_CRYSTAL`
    
    - Crystal Report 뷰
    
    ### `MC_MB_VIEW`
    
    - View 메뉴 버튼 그룹
    
    ### 10. Variant (레이아웃 변형)
    
    ### `MC_FC_LOAD_VARIANT`
    
    - Variant 불러오기
    
    ### `MC_FC_SAVE_VARIANT`
    
    - Variant 저장
    
    ### `MC_FC_MAINTAIN_VARIANT`
    
    - Variant 관리
    
    ### `MC_FC_CURRENT_VARIANT`
    
    - 현재 Variant 정보
    
    ### `MC_MB_VARIANT`
    
    - Variant 메뉴 그룹
    
    ### 11. 출력(Print) 계열
    
    ### `MC_FC_PRINT`
    
    - 출력
    
    ### `MC_FC_PRINT_PREV`
    
    - 출력 미리보기
    
    ### `MC_FC_PRINT_BACK`
    
    - 출력 후 돌아가기
    
    ### `MC_FC_PRINT + _BACK`
    
    - 출력 관련 전체 흐름 제어
    
    ### 12. 내보내기 / 외부 연동 (Export)
    
    ### `MC_FC_CALL_XXL`
    
    - Excel (XXL)
    
    ### `MC_FC_CALL_XML_EXPORT`
    
    - XML Export
    
    ### `MC_FC_PC_FILE`
    
    - PC 파일 저장
    
    ### `MC_FC_SEND`
    
    - SAP 내부 전송
    
    ### `MC_FC_TO_OFFICE`
    
    - Office 연동
    
    ### `MC_FC_HTML`
    
    - HTML 출력
    
    ### `MC_FC_URL_COPY_TO_CLIPBOARD`
    
    - URL 복사
    
    ### `MC_MB_EXPORT`
    
    - Export 메뉴 그룹
    
    ### 13. 그래프
    
    ### `MC_FC_GRAPH`
    
    - ALV 데이터를 그래프로 표시
    
    ### 14. Function Group 단위 제어 (중요)
    
    ### `MC_FG_EDIT`
    
    - 편집 관련 버튼 전체
    
    ### `MC_FG_SORT`
    
    - 정렬 관련 버튼 전체
    
    ### `MC_FG_VIEW`
    
    - View 관련 버튼 전체
    
    ### `MC_FC_EXCL_ALL`
    
    - **모든 표준 버튼 제거**
        
        → 커스텀 버튼만 쓰고 싶을 때 사용
        
    
    ### 15. 실무에서 제일 많이 쓰는 조합
    
    ### 1) 출력/엑셀 막기
    
    ```abap
    APPEND cl_gui_alv_grid=>mc_mb_export TO lt_exclude.
    
    ```
    
    ### 2) 편집 기능 전부 막기
    
    ```abap
    APPEND cl_gui_alv_grid=>mc_fg_edit TO lt_exclude.
    
    ```
    
    ### 3) 전부 막고 커스텀만
    
    ```abap
    APPEND cl_gui_alv_grid=>mc_fc_excl_all TO lt_exclude.
    
    ```
    
    ### 16. 핵심 요약 (이거만 기억해도 됨)
    
    ```
    MC_FC_*  : 개별 버튼
    MC_MB_*  : 메뉴 버튼(그룹)
    MC_FG_*  : 기능 그룹 전체
    MC_FC_EXCL_ALL : 전체 제거
    
    ```
